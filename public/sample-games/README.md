# Summit Sprint (Celeste-style clone)

This project is a small browser game inspired by Celeste movement.

## Included mechanics

- Run with acceleration and air control
- Coyote time + jump buffering
- Variable jump height
- Wall slide + wall jump
- Air dash (single dash, crystal refill)
- Spikes, berries, checkpoints, respawn, summit goal

## Run it

Open `index.html` in a browser, or serve the folder with a static server:

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Controls

- Move: Arrow keys or `A`/`D`
- Jump: `Space`, `Z`, or `C`
- Dash: `X` or `Shift`
- Restart run: `R` or `Enter`

Touch controls appear automatically on mobile devices.
